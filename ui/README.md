# Agent Control UI

The Agent Control UI is a Next.js dashboard for managing agents, controls, and monitoring data from your Agent Control server. It provides a visual workflow for creating and editing controls, inspecting evaluation activity, and tracking pass/block metrics without touching code.

## What you can do

- Create, enable, disable, and edit controls
- Associate controls with agents and review active configurations
- Monitor evaluations, denials, and recent activity in real time
- Inspect evaluator types and configuration details in one place

## Quick start

From the repo root:

```bash
cd ui
pnpm install
pnpm dev
```

The UI runs on http://localhost:4000 and expects the server on http://localhost:8000 by default.

Full guide: https://docs.agentcontrol.dev/components/ui
